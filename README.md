# Histórias de Usuário

## Cliente

- Como cliente, quero criar um pedido de aluguel informando veículo e dados financeiros para solicitar um contrato de locação.
- Como cliente, quero selecionar um veículo disponível para compor meu pedido de aluguel.
- Como cliente, quero informar meus dados financeiros para permitir a análise de crédito do pedido.
- Como cliente, quero editar um pedido enquanto ele estiver pendente para corrigir informações antes da análise.
- Como cliente, quero cancelar um pedido para evitar continuidade de análise ou contratação.
- Como cliente, quero visualizar o status do pedido para entender se foi aprovado ou reprovado.
- Como cliente, quero registrar uma conta para poder acessar o sistema de gestão de aluguel.
- Como cliente, quero me autenticar no sistema para consultar e gerenciar meus pedidos com segurança.

## Agente

- Como agente, quero analisar pedidos de aluguel com base nos dados do cliente para decidir sobre concessão de crédito.
- Como agente, quero consultar os dados do cliente durante a análise para fundamentar a decisão de crédito.
- Como agente, quero validar o risco financeiro de um pedido para reduzir inadimplência na contratação.
- Como agente, quero aprovar um pedido elegível para dar continuidade à formalização do contrato.
- Como agente, quero reprovar um pedido que não atenda aos critérios de crédito para evitar risco financeiro.
- Como agente, quero emitir um parecer financeiro para formalizar a decisão de aprovação ou reprovação.
